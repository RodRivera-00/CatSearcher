# Cat Searcher

Your easy way to search and adore cats!

## Setup Instructions

### Pull the repository

#### `git pull https://github.com/Yeonin/CatSearcher.git`

### Go to the project directory

 `cd CatSearcher`

### Install pre-requisites

 `npm install`

### Run the project

`npm start`

### Visit the url

`http://locahost:3000`
